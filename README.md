ChatGen AI - Smart AI Chatbot
📌 Project Overview
ChatGen AI is a full-stack AI-powered chatbot application built using the MERN stack. The application allows users to register, log in, and interact with an AI assistant in real time. It integrates Google's Gemini AI model to generate intelligent responses and stores user data securely using MongoDB.

🚀 Features
User Authentication (Login & Signup)
JWT-Based Authorization
AI Chat using Google Gemini API
Real-Time Messaging
Responsive User Interface

⚙️ Installation
Clone Repository
git clone

Install Backend Dependencies
cd backend

npm install

Install Frontend Dependencies
cd frontend

npm install

Configure Environment Variables
Create a .env file:

PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

GEMINI_API_KEY=your_gemini_api_key

GEMINI_MODEL=gemini-2.0-flash

Run Backend
npm run dev

Run Frontend
npm run dev
